# 🛡️ ViroScan
### Web-Based Malware & Threat Detection Platform for URLs and Files

ViroScan is a cybersecurity-focused web application developed as a **BCA Final Year Project** under **Tilak Maharashtra Vidyapeeth, Pune**.  
The platform helps users detect malicious URLs and infected files using real-time threat analysis.

---

## 🚀 Features

- 🔍 URL Malware & Phishing Detection
- 📁 File Scanning for Threats
- 🛡️ VirusTotal API Integration
- 📊 Scan History Dashboard
- 🚨 Threat Reporting System
- 👨‍💼 Admin Panel
- 🌐 Browser Extension Support
- 📱 Responsive User Interface

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Backend:** Python, Flask
- **Database:** SQLite
- **API:** VirusTotal API
- **Extension:** Chrome/Firefox Browser Extension

---

## 📌 Project Modules

### 1. URL Scanner
Scans suspicious URLs and detects phishing or malicious websites.

### 2. File Scanner
Allows users to upload files and checks them for malware threats.

### 3. Scan History
Stores previous scan records for users.

### 4. Threat Reporting
Users can report malicious links or files for admin review.

### 5. Admin Panel
Manages reports, users, and monitoring activities.

### 6. Browser Extension
Provides real-time URL scanning directly from the browser.

---

## 🎯 Objective

The main objective of ViroScan is to provide users with a simple and effective platform to identify malicious websites and infected files, helping improve cybersecurity awareness and safe browsing practices.
